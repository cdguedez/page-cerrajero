# Cerrajería Express Caracas

Página web (Landing Page) para **Cerrajería Express Caracas**, un servicio de cerrajería a domicilio disponible 24/7 en la ciudad de Caracas, Venezuela.

## Características

- Diseño moderno, responsivo y optimizado para SEO local.
- Integración directa con WhatsApp para contacto inmediato.
- Secciones de servicios (Apertura de Puertas, Cerrajería Automotriz, Cambio de Cilindros).

## Tecnologías Utilizadas

- HTML5 (Semántico)
- CSS3 (Vanilla, Custom properties, animaciones)
- JavaScript (Módulos ES6)
- [Vite](https://vitejs.dev/) como herramienta de construcción (Build Tool).

## Requisitos Previos

Asegúrate de tener instalado [Node.js](https://nodejs.org/) en tu sistema.

## Instalación y Desarrollo Local

1. Clona este repositorio o descarga los archivos.
2. Navega al directorio del proyecto en tu terminal:
   ```bash
   cd page
   ```
3. Instala las dependencias:
   ```bash
   npm install
   ```
4. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
5. Abre la URL local (por defecto `http://localhost:5173`) en tu navegador para ver la página.

## Scripts Disponibles

- `npm run dev`: Inicia el servidor de desarrollo local de Vite.
- `npm run build`: Construye la versión optimizada para producción.
- `npm run preview`: Previsualiza la versión construida localmente.
